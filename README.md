# 🛡️ PhishGuard

## 1. Project Overview

* A phishing email detection web application.
* Analyzes email content to estimate phishing risk.
* Displays the phishing probability as a percentage.

## 2. Features

* Email Content Analysis
* Phishing Risk Percentage
* Predefined Detection Logic
* Instant Analysis Result
* User-Friendly Interface
* Responsive Design

## 3. Technologies Used

* HTML5
* CSS3
* JavaScript


## 4. How It Works

1. Enter or paste the email content.
2. Click the **Analyze** button.
3. The system checks the email using predefined rules.
4. A phishing risk percentage is displayed.
5. Users can determine whether the email appears safe or suspicious.

## 5. Purpose

* Raise awareness about phishing emails.
* Help users understand potential phishing attempts.
* Demonstrate phishing detection using simple logic.

## 6. Future Improvements

* AI-powered email analysis.
* Real-time phishing detection.
* URL reputation checking.
* Machine Learning integration.
* Email attachment scanning.
